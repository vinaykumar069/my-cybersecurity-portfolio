# Vinay's Offensive Security Portfolio

A dark-themed, terminal-inspired portfolio website for an offensive security professional.

## Features
- **Interactive Terminal**: A fully functional simulated terminal.
- **Particle Background**: Dynamic background effects.
- **Glitch Animations**: Cyberpunk/hacker aesthetic.
- **Responsive Design**: Mobile-friendly layout.
- **Project Showcase**: Display of security projects and tools.

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Three.js (for 3D effects)

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
